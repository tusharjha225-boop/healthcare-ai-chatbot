# healthcare-ai-chatbot
NLP-based Healthcare AI Chatbot built using Google Gemma LLM and Gradio to provide safe and informative health-related responses.
## Features
- Answers basic healthcare-related questions
- FAQ-based instant responses
- AI-powered contextual conversation
- Safe responses (No medical prescriptions)
- User-friendly interface using Gradio

## Technologies Used
- Python
- Natural Language Processing (NLP)
- Google Gemma LLM
- Gradio
- Machine Learning

## Project Objective
The objective of this project is to develop a healthcare assistant chatbot that can help users with basic health-related information and guide them to consult healthcare professionals when necessary.

## How to Run
1. Install Python
2. Install required libraries using requirements.txt
3. Run the chatbot.py file
4. Open the Gradio link in browser
